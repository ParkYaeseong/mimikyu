Projects & Achievements

AI 기반 임상의사결정 시스템(CDSS) 개발을 총괄하며 이뤄낸 주요 프로젝트와 성과입니다.

🚀 MEORING CDSS: 환자 맞춤형 다중모달리티 임상 의사결정 지원 시스템

2025.05 ~ 2025.06 (팀 프로젝트)

역할: 프로젝트 리더(PL) & 풀스택 개발자

개요: 의료진이 EMR, 다중오믹스, CT 영상 등 분산된 의료 데이터를 통합 분석하고, AI 기반의 암 진단 및 예후 예측 결과를 임상 현장에서 효과적으로 활용할 수 있는 웹 기반 CDSS (Clinical Decision Support System)를 구축하는 것이 목표. 최종적으로는 AI 분석 결과를 OpenEMR 등 기간계 시스템과 연동하여 진료 워크플로우에 통합하는 것을 목표로 함.

주요 성과:

  • EHR 연동 풀스택 시스템 아키텍처 설계: OpenEMR, LIS, PACS 데이터를 연동하고 Django(Python) 백엔드와 React/Flutter 프론트엔드를 연결하는 전체 시스템 아키텍처 설계.
  • AI 오케스트레이터 설계 및 비동기 처리 도입: 다양한 AI 모델 호출, 데이터 전처리, 결과 반환을 관리하는 AI 오케스트레이터 설계. Celery를 활용한 비동기 처리 방식으로 대규모 분석 효율성 확보.
  • Python/R 기반 분석 및 웹/모바일 애플리케이션 개발: Django REST Framework로 핵심 도메인 API 서버 구축 및 역할 기반 접근 제어(RBAC) 구현. React로 다중모달리티 데이터 분석 결과 대시보드 개발 및 XAI 분석 결과 시각화(SHAP, Grad-CAM) 구현.
  • 생성형 AI 및 통계 알고리즘 연동 개발: 췌장암, 유방암, 난소암, 간암, 폐암 등 진단/예후 예측 AI 모델 개발 및 백엔드 API 연동. RAG 기술과 Gemini API 활용 의료 챗봇 구현.
  • 규칙 기반 약물 상호작용 엔진 개발: 국내 약물 API와 연동하여 약물 상호작용 및 안전성을 검토하는 엔진 개발 및 OCS 연동.
  • 엔드투엔드 프로젝트 관리 및 기술 리더십: 데이터 수집/정제부터 모델 학습/평가, XAI 기반 결과 해석까지 프로젝트 전 과정을 PL로서 주도적으로 수행하며, 팀원들의 기술 성장을 지원.

About Me

"유전적으로 동일한데 왜 다를까?" 어린 시절 쌍둥이 동생을 보며 품었던 질문은 생명 현상에 대한 호기심으로 이어졌고, 시스템생명공학과 진학의 계기가 되었습니다. 학부에서는 유전체부터 대사체까지 생명 시스템을 통합적으로 이해하는 학문을 배우며, 복잡한 생명 현상을 규명하기 위한 첫 단추는 바로 ‘데이터’에 있음을 깨달았습니다.

졸업 후 을지대학교병원과 아산병원에서 근무하며, 유전체·임상·의료영상 등의 데이터가 분절된 '데이터 사일로' 문제를 직접 경험했습니다. 특히 CT, MRI 등 방대한 DICOM 영상 데이터를 수작업으로 처리하며, AI를 통한 자동화와 정밀한 정량화의 필요성을 절감했습니다. 이 경험은 제가 질병의 조기 진단을 목표로 하는 의료 AI 연구에 뛰어들게 된 결정적 계기가 되었습니다.

  • 풀스택 개발 역량: Django REST Framework 기반 백엔드 API 개발, React/Flutter 기반 프론트엔드 개발, Docker 기반 배포 경험.
  • 임상 및 생물학적 도메인 지식: 세포/동물 실험, 인체유래물 관리 경험을 통한 의료/생물학 분야에 대한 깊은 이해.
